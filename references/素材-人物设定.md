---

## 一、核心框架：思想 → 性格 → 命运

人物设定的本质是一条因果链：

```
思想（核心信念）
  ↓ 决定
性格（行为模式）
  ↓ 导致
命运（故事走向）
```

- **思想**：角色最核心的信念/执念/恐惧
- **性格**：基于思想形成的行为倾向和处事风格
- **命运**：性格在具体环境中的必然结果

---

## 二、三项评估指标

| 指标 | 含义 | 评估方式 |
|:-----|:-----|:---------|
| **合理** | 人物行为是否符合其设定的性格和动机 | 行为是否有内在逻辑，是否自洽 |
| **有特点** | 是否有区别于同类角色的独特之处 | 一句话能否描述这个角色的核心特征 |
| **有代入感** | 读者是否能够理解和共情 | 读者的困境/欲望是否能投射到角色上 |

> 详见 `character_template.py` 的 `EVALUATION_METRICS` 常量。

## 四、配角模板

功能型配角的设计要点与常见失误，详见 `character_template.py` 的 `SUPPORTING_TEMPLATES` 常量：

| 配角类型 | 核心功能 | 设计要点 | 常见失误 |
|:---------|:---------|:---------|:---------|
| 引路人 | 推动剧情、提供信息 | 必须有自己的动机 | 只当工具人无自主性 |
| 竞争对手 | 制造冲突、衬托主角 | 实力相近有来有回 | 单纯的恶人无深度 |
| 情感锚点 | 让读者关心 | 有独立的成长空间 | 纯花瓶无主动性 |
| 搞笑担当 | 调节气氛 | 不抢主线节奏 | 喧宾夺主 |
| 背叛者 | 制造转折 | 要有合理的背叛原因 | 无铺垫的突然背叛 |
| 导师 | 指引成长 | 适时退出舞台 | 一直帮主角解决一切 |

### 配角数量建议

| 篇幅规模 | 建议主要配角数 |
|:---------|:-------------|
| 短篇（5万字以内）| 3-5 个 |
| 中篇（5-20万字）| 5-8 个 |
| 长篇（20万字+）| 8-15 个（分批引入）|

> 详见 `character_template.py` 的 `SUPPORTING_COUNT` 常量。

## 五、人设模板

创建人设时需填写的字段，详见 `character_template.py` 的 `CHARACTER_SHEET_FIELDS` 常量：

```
角色名：
身份/职业：
核心思想：（一句话描述最深层的信念/恐惧）
性格关键词：（3个）
外在特征：（外貌/习惯/口癖）
内在矛盾：（最核心的内心冲突）
关系网：（与谁有什么样的关系）
角色功能：（在故事中承担什么功能）
成长弧线：（从哪里开始，到哪里结束）
绝不开口/做的事：（角色的底线）
一句话速写：（用一句话让读者记住这个角色）
```

## 六、使用方式

```bash
# 查询某类型的人物原型
python scripts/character_template.py "末世重生"

# 仅查询主角原型
python scripts/character_template.py "玄幻" --role protagonist

# 仅查询配角模板
python scripts/character_template.py "都市" --role supporting
```
