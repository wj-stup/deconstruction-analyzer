# 素材·开篇技法

> 开篇写作原则与自检指南（人类可读参考）
> 详细数据见 `scripts/opening_formula.py`
> 更新时间：2026-06-25

---

## 一、黄金 300 字法则

- **前 50 字**：必须出现画面感（场景/动作/感官）
- **前 100 字**：读者必须知道主角是谁
- **前 200 字**：必须出现异常（与日常不同的信号）
- **前 300 字**：必须给出"为什么要继续读"的理由（悬念/冲突/反差）

---

## 二、五种有效开篇类型

| 类型 | 核心手法 | 适用类型 | 示例 |
|:-----|:---------|:---------|:-----|
| 冲突型 | 以矛盾/对抗开场 | 玄幻、都市 | 被退婚、被追杀、被陷害 |
| 身份重置型 | 主角身份剧变 | 末世重生、穿越 | 醒来发现重生/穿越了 |
| 金手指展示型 | 直接展示核心设定 | 系统流、异能 | 系统面板浮现、异能觉醒 |
| 悬念型 | 抛出谜题驱动阅读 | 悬疑、推理 | 一封神秘来信、一桩离奇事件 |
| 日常打破型 | 平静日常被打破 | 都市、科幻 | 普通一天突然出现异常 |

> 各公式的详细结构/节奏/适用范围，见 `opening_formula.py` 的 `OPENING_FORMULAS` 常量。

---

## 三、开篇节奏控制

- 短句优先：开篇前 500 字平均句长不超过 20 字
- 动作驱动：用动作推进而非内心独白
- 信息渐进：不要在前三章倾倒世界观设定

---

## 四、易犯错误（12 条）

详见 `opening_formula.py` 的 `COMMON_MISTAKES` 常量，核心要点：

1. **大段世界观铺陈** — 开篇就写设定集，读者直接流失
2. **主角无目标** — 没有明确的行动方向
3. **慢热过度** — 超过 3 章还没有任何冲突或悬念
4. **信息过载** — 前 3 章引入太多角色和组织
5. **被动主角** — 主角只是被动接受事件
6. **平淡日常** — 开篇写了一堆日常但没有任何异常信号
7. **频繁切换视角** — 开篇就在多个角色之间跳转
8. **过度内心独白** — 大段主角的心理活动
9. **预告式叙事** — 展示而非预告
10. **开篇就虐** — 没有建立情感联结就安排惨烈桥段
11. **缺乏钩子** — 每章结尾平滑过渡
12. **千人一面** — 开篇套路与其他作品雷同

## 五、自检清单

详见 `opening_formula.py` 的 `SELF_CHECK_LIST` 常量：

```
□ 前3句话是否出现了冲突/悬念/反差？
□ 第300字内是否交代了"与众不同"？
□ 第800字内是否有第一个小冲突？
□ 第一章结尾是否有悬念/钩子？
□ 开篇是否有"可记忆点"？
□ 对标作品中最高赞的开篇用了什么手法？
□ 主角是否有明确的短期目标？
□ 前3章是否建立了至少1个读者关心的问题？
□ 信息释放节奏是否渐进（非一次性倾倒）？
□ 开篇是否与该类型的读者预期匹配？
```

## 六、各类型开篇偏好

| 类型 | 推荐公式 | 避免事项 |
|:-----|:---------|:---------|
| 玄幻 | A 或 B | 金手指太早会压缩悬念空间 |
| 末世重生 | B（醒来最自然）| 日常太长后再切入 |
| 悬疑推理 | D（悬念型）| 大段背景交代 |
| 都市 | A 或 C | 玄幻式长篇设定 |
| 言情 | 冲突型或日常打破 | 大段心理描写 |
| 科幻 | E 或 D | 科普式叙事 |
| 修仙/修真 | B 或 A | 金手指太早压缩悬念 |
| 游戏 | C 或 A | 慢热日常 |
| 历史/架空历史 | B 或 A | 大段历史科普 |
| 轻小说/二次元 | A 或 C | 沉重开头 |

> 完整的类型→公式映射（含避开原因），见 `opening_formula.py` 的 `TYPE_FORMULA_MAP`。
