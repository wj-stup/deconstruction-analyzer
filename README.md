# deconstruction-analyzer
网文拆书分析助手，专注爆款网文拆书分析，输出结构化拆书报告。内置11大类·33小类·84子类三级分类体系，按类型自动路由分析维度（如末世重生侧重蝴蝶效应/囤货逻辑，悬疑推理侧重线索铺设/反转设计，无限流生存游戏侧重规则体系/高压节奏，伪人末日侧重信任博弈/身份认知）。支持交叉类型自动识别。可独立使用分析网文类型/作品，也可与 multi-role-novel-iteration 握手配合，为多角色创作天团提供拆书参考。新增写作问题诊断功能——当用户说"节奏太慢""开篇没人看"时，自动匹配症状→根因→方案。当用户说"帮我拆一下XX类""分析爆款结构""拆书""网文拆解""拆解这个类型""分析XX小说"时触发。
