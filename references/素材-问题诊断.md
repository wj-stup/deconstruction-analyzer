# 素材·问题诊断

> 12 类写作问题的症状 → 根因 → 方案映射表
> 用于写作过程中断点续命
> 更新时间：2026-06-22

---

## 使用方法

当用户说"我卡住了""节奏太慢""开篇没人看"等写作问题时：
1. 解析用户描述 → 提取关键词
2. 匹配下表中的症状关键词
3. 按对应的"可能根因"逐项排查
4. 输出根因分析 + 解决方案

## 问题诊断映射表

### 问题 1：开篇没人看

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 开篇没人看、开头无聊、前几章没吸引力、读者不往下翻 |
| **可能根因** | ①大段设定铺陈②没有冲突/悬念③主角没有目标④节奏太慢⑤千人一面 |
| **排查问题** | 前300字有异常信号吗？前3章有冲突吗？主角有明确目标吗？ |
| **解决方案** | 用冲突/重置型开篇重写第一章；把设定后置到行动中带出；添加开篇钩子 |
| **市场对标** | 参考拆书报告中"开篇钩子公式"章节，对照最高赞开篇模式 |

### 问题 2：节奏太慢

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 节奏慢、拖沓、流水账、读着没劲、太平了 |
| **可能根因** | ①铺垫过长②缺乏爽点③信息密度不均④时间流速失当 |
| **排查问题** | 连续几章有无新事件？爽点间隔多少章？有无连续3章无冲突？ |
| **解决方案** | 压缩铺垫段3:1原则；每3-5章安排1个小爽点；快慢交替节奏 |
| **市场对标** | 参考拆书报告中"爽点节奏图"的情绪曲线模板 |

### 问题 3：角色不鲜活

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 角色脸谱化、不像真人、配角工具人、主角没特点 |
| **可能根因** | ①缺少核心信念②行为缺乏内在逻辑③没有矛盾面④全是功能型配角 |
| **排查问题** | 能一句话说清角色核心信念吗？有"绝对不会做的事"吗？配角有独立动机吗？ |
| **解决方案** | 用"思想→性格→命运"框架重设角色；给主角加矛盾面；让配角有自主性 |
| **市场对标** | 参考拆书报告中"人物原型库"的主角原型对照表 |

### 问题 4：金手指不合理

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 金手指太强、开后开太大、没有代价、开挂感太强 |
| **可能根因** | ①金手指无限制②无成长感③违背公平性④与世界观不兼容 |
| **排查问题** | 金手指有明确限制吗？使用有代价吗？升级路径清晰吗？ |
| **解决方案** | 添加使用限制和代价；设置升级条件；让金手指与世界观规则绑定 |
| **市场对标** | 参考拆书报告中"金手指类型库"的设计原则 |

### 问题 5：逻辑漏洞

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 逻辑不自洽、前后矛盾、设定冲突、蝴蝶效应滥用 |
| **可能根因** | ①设定未通盘考虑②遗忘前文伏笔③金手指规则不一致 |
| **排查问题** | 重新通读全文，标记每个设定规则；检查是否有规则被后来打破 |
| **解决方案** | 创建设定文档统一管理规则；修复矛盾点；必要时回铺前文伏笔 |
| **市场对标** | 参考拆书报告中"常见雷区"的逻辑漏洞类目 |

### 问题 6：爽点不够爽

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 爽点不爽、高潮不起来、打脸没感觉、压抑后没释放 |
| **可能根因** | ①铺垫不足②爽点太短③预期太容易猜到④反差不够 |
| **排查问题** | 爽点前有多少铺垫？爽点持续几段？读者能提前猜到吗？ |
| **解决方案** | 加长铺垫+缩短爆发=更大反差；设置出人意料的转折；爽点后留回味 |
| **市场对标** | 参考拆书报告中"爽点节奏图"的爽点类型分布 |

### 问题 7：卡文写不动

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 卡文、写不下去、不知道接下来写什么、没有灵感 |
| **可能根因** | ①大纲缺失②没有短期目标③写了太多没规划④走到死胡同 |
| **排查问题** | 有完整大纲吗？下一章的核心事件是什么？主线方向清晰吗？ |
| **解决方案** | 补充细纲（下5章的核心事件）；回到大纲看方向；用"如果...会怎样"引出新事件 |
| **市场对标** | 参考拆书报告中"结构模板"的全篇蓝图 |

### 问题 8：读者弃书

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 弃书率高、追读下降、完本率低、后面没人看 |
| **可能根因** | ①中段无聊②承诺未兑现③节奏单一④持续压抑无释放 |
| **排查问题** | 弃书集中发生在哪个位置？是否有长时间无爽点段？ |
| **解决方案** | 在弃书高峰点添加转折；确保每个Act有1-2个记忆点；张弛交替 |
| **市场对标** | 参考拆书报告中"爽点节奏图"的节奏对标表 |

### 问题 9：感情线突兀

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 感情线生硬、CP没有化学反应、相爱太突然、感情不自然 |
| **可能根因** | ①缺少互动铺垫②没有阻碍③没有互补性④纯功能型配对 |
| **排查问题** | CP有共同经历吗？有合理阻碍吗？性格是互补还是冲突？ |

| **市场对标** | 参考拆书报告中"人物原型库"的主角原型对照表 |

### 问题 10：世界观混乱

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 设定太多、世界观讲不清、读者看不懂、设定打架 |
| **可能根因** | ①一次性释放太多设定②设定间互相矛盾③没有体系化 |
| **排查问题** | 第一章新设定超过3个吗？读者能一句话描述世界规则吗？ |
| **解决方案** | 分批释放设定（用到才说）；创建设定词典统一管理；用事件展示规则 |
| **市场对标** | 参考拆书报告中"类型总览"的核心类型特征 |

### 问题 11：结尾无力

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 结尾太平、高潮不起来、结尾草率、读者不满意 |
| **可能根因** | ①高潮前铺垫不够②核心冲突未彻底解决③情感线未闭合 |
| **排查问题** | 主角的目标达成了吗？核心冲突有明确胜负吗？各辅线有闭环吗？ |
| **解决方案** | 在高潮前设置"最低点"增强反差；确保核心冲突的终极对决；闭合所有辅线 |
| **市场对标** | 参考拆书报告中"结构模板"的Act 3占比建议 |

### 问题 12：同质化严重

| 项 | 内容 |
|:---|:-----|
| **症状关键词** | 和别人写的差不多、没有新意、套路感太重、千篇一律 |
| **可能根因** | ①盲目跟风②没有创新点③照搬模板④缺乏个人特色 |
| **排查问题** | 你的故事和同类作品的核心差异是什么？有同类没尝试过的元素吗？ |
| **解决方案** | 在经典框架上加入1-2个创新变体；交叉类型带来新鲜感；找到独特视角 |
| **市场对标** | 参考拆书报告中"常见雷区"的创新机会分析 |
