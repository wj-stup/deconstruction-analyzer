## 三、密度建议

按目标字数的爽点密度配置，详见 `thrillpoint_lookup.py` 的 `DENSITY_ADVICE`：

| 字数区间 | 总章节数 | 小爽点间隔 | 大爽点间隔 |
|:---------|:---------|:-----------|:-----------|
| 2万字 | 10-15章 | 每2-3章 | 每5-8章 |
| 5万字 | 25-35章 | 每3-5章 | 每8-12章 |
| 10万字 | 50-70章 | 每3-5章 | 每10-15章 |
| 20万字 | 100-130章 | 每5-8章 | 每15-20章 |

---

## 四、使用原则

1. **密度控制**：每 3-5 章安排一个小爽点，每 10-15 章安排一个大爽点
2. **类型交替**：不要连续使用同一类型的爽点，避免审美疲劳
3. **与类型匹配**：末世文侧重"收获类"、玄幻文侧重"升级类"、都市文侧重"打脸类"
4. **铺垫很重要**：爽点效果取决于铺垫质量，3 分铺垫 7 分回报
5. **创新空间**：在经典爽点基础上加入该类型的创新变体，增加新鲜感

## 五、使用方式

```bash
# 查询某类型的侧重爽点
python scripts/thrillpoint_lookup.py "末世重生"

# 查询指定类别
python scripts/thrillpoint_lookup.py "玄幻" --category "逆袭类"

# 列出所有类别和条目
python scripts/thrillpoint_lookup.py --list-categories
```
