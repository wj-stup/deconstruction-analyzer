# 网文类型分类体系

> **人类可读摘要**。权威数据源为 `taxonomy_data.py` 的 `TAXONOMY` / `SYNONYM_MAP` / `WEIGHT_MATRIX` 常量，额外维度见 `TAXONOMY` 各子类的 `extra_dimensions` 字段。
> 本文件保留分类总览和关键映射，详细子类定义（ID/关键词/额外维度）以代码为准。

## 额外维度映射

| 感情线节奏 | 言情 |
| cp感 | 言情 |
| 科技设定 | 科幻 |
| 历史考据 | 历史/架空历史 |
| 脑洞/梗密度 | 轻小说/二次元 |
| 规则体系 | 无限流/生存游戏 |
| 生存逻辑 | 无限流/生存游戏、末世/废土 |
