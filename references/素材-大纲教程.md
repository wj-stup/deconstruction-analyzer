> 大纲构成方法与结构设计指南（人类可读参考）
> 详细数据见 `scripts/outline_generator.py`
> 更新时间：2026-06-25

---

## 一、大纲构成要素

### 1.1 核心要素

| 要素 | 说明 | 缺失后果 |
|:-----|:-----|:---------|
| 主角目标 | 主角想要达到什么 | 故事没有方向 |
| 核心冲突 | 阻碍主角达到目标的力量 | 缺乏戏剧张力 |
| 主线事件 | 从起点到终点的主要路径 | 情节散漫 |
| 世界观框架 | 故事发生的规则和环境 | 设定不自洽 |
| 主题/立意 | 故事想表达什么 | 缺乏深度 |

### 1.2 三幕结构

```
Act 1: 开局（约 20-25%）
├── 日常展示 → 异常出现 → 冒险召唤 → 出发
└── 核心任务：建立角色、世界、冲突

Act 2: 发展（约 50-55%）
├── 试炼 → 中点转折 → 最低点
└── 核心任务：升级冲突、揭示真相、压抑到极致

Act 3: 高潮（约 20-25%）
├── 反击 → 最终决战 → 新平衡
└── 核心任务：解决核心冲突、完成角色弧线
```

## 二、主线设计

### 2.1 主线类型

| 类型 | 驱动力 | 适用 | 代表模式 |
|:-----|:-------|:-----|:---------|
| 目标驱动 | 主角要达成某个目标 | 玄幻、都市 | 升级→夺冠→飞升 |
| 生存驱动 | 主角必须在危险环境中活下来 | 末世、悬疑 | 生存→反击→安全 |
| 真相驱动 | 主角要揭开某个秘密 | 悬疑、推理 | 困惑→线索→真相 |
| 情感驱动 | 主角要与某人建立/修复关系 | 言情 | 相遇→阻碍→在一起 |
| 成长驱动 | 主角要完成自我蜕变 | 都市、修仙 | 挫折→领悟→蜕变 |

### 2.2 主线节奏规划

| 章节区间 | 节奏 | 关键节点 |
|:---------|:-----|:---------|
| 1-3 章 | 启动 | 开篇钩子 + 核心冲突亮相 |
| 4-8 章 | 加速 | 第一个小高潮 + 金手指/核心设定全面展示 |
| 9-15 章 | 波峰 | 中点大转折 |
| 16-22 章 | 蓄力 | 最低点 + 触底反弹 |
| 23-30 章 | 冲刺 | 最终决战 + 结局 |

## 三、辅线设计

### 3.1 辅线类型

详见 `outline_generator.py` 的 `SUBPLOT_TYPES` 常量：

| 辅线类型 | 功能 | 占比建议 | 引入时机 |
|:---------|:-----|:---------|:---------|
| 感情线 | 丰富角色维度、增加情感投入 | 15-25% | Act 1 后半 |
| 竞争线 | 制造对手压力、驱动进步 | 10-20% | Act 1 末尾 |
| 解谜线 | 维持好奇心、增加层次感 | 10-15% | 贯穿全文 |
| 成长线 | 展示角色内在变化 | 10-15% | 与主线交织 |
| 副副本线 | 提供小闭环、调节节奏 | 5-10% | 张弛之间的缓冲 |

### 3.2 辅线与主线关系

- **平行型**：辅线独立推进，在关键节点与主线交汇
- **交织型**：辅线事件直接影响主线走向
- **暗线型**：表面看不到，到中段/结尾才揭示与主线的关联

### 3.3 辅线设计禁忌

1. 辅线喧宾夺主：超过主线篇幅
2. 辅线无闭环：引入了但没解决
3. 辅线与主线无关：完全独立的副本对主线无影响
4. 辅线过多：同时 3+ 条辅线导致焦点涣散

## 四、各类型三幕大纲模板

11 种网文类型的详细三幕结构模板（含各 Act 名称、占比、主线/辅线内容），详见 `outline_generator.py` 的 `OUTLINE_TEMPLATES` 常量。

### 已覆盖类型

| 类型 | Act 1 | Act 2 | Act 3 |
|:-----|:------|:------|:------|
| 玄幻 | 起步期 | 磨砺期 | 爆发期 |
| 修仙/修真 | 起步期 | 磨砺期 | 飞升期 |
| 都市 | 转折点 | 上升期 | 峰值期 |
| 末世/废土 | 囤货期 | 生存期 | 反击期 |
| 科幻 | 发现期 | 探索期 | 决战期 |
| 悬疑/推理 | 入局期 | 调查期 | 揭秘期 |
| 游戏 | 新手期 | 成长期 | 巅峰期 |
| 历史/架空历史 | 入世期 | 经营期 | 霸业期 |
| 言情 | 相遇期 | 纠葛期 | 团圆期 |
| 轻小说/二次元 | 引入期 | 展开期 | 高潮期 |

> 运行 `python scripts/outline_generator.py "玄幻" --words 50000` 可查看某类型的完整大纲模板（含字数估算）。

## 五、分集节奏建议

详见 `outline_generator.py` 的 `WORD_COUNT_MAP` 常量：

| 字数区间 | 建议章节数 | 每章字数 | 爽点间隔 |
|:---------|:---------|:---------|:---------|
| 2万字 | 10-15章 | 1500-2000字 | 每2-3章 |
| 5万字 | 25-35章 | 1500-2000字 | 每3-5章 |
| 10万字 | 50-70章 | 1500-2000字 | 每5-8章 |
| 20万字 | 100-130章 | 1500-2000字 | 每5-8章 |
| 50万字+ | 长篇连载 | 2000-3000字 | 每8-12章 |

## 六、使用方式

```bash
# 生成某类型的大纲模板（默认2万字）
python scripts/outline_generator.py "末世重生"

# 指定字数
python scripts/outline_generator.py "玄幻" --words 50000
```
